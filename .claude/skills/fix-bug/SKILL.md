# fix-bug

## Usage

```
/fix-bug
```

## Workflow

1. 使用 Skill 工具加载 `systematic-debugging` skill，先找到根因
2. 使用 Skill 工具加载 `test-driven-development` skill，先写失败测试
3. 最简修复，禁止过度设计
4. 提交修复和回归测试
5. 完成后自动写 failure observation
